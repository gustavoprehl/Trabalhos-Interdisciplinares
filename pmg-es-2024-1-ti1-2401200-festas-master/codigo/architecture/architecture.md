# Arquitetura do Projeto

Este documento descreve a arquitetura geral do projeto, incluindo a organização dos diretórios e sua finalidade.

## Estrutura de Diretórios

O projeto segue a seguinte estrutura de diretórios:

- **pages**: Este diretório contém todas as páginas do site. Cada página é representada por um arquivo HTML, organizado de forma a facilitar a navegação e manutenção do projeto.

- **style**: Aqui são armazenados os arquivos relacionados aos estilos do site.
  - **global.css**: Este arquivo contém os estilos globais que serão aplicados em todo o projeto. Qualquer estilo que deva ser compartilhado entre diferentes partes do site deve ser definido aqui.

- **js**: Este diretório contém os arquivos JavaScript utilizados no projeto. Cada arquivo JavaScript pode ser responsável por funcionalidades específicas do site, como interações de usuário, validações de formulários, entre outros.

- **assets**: Este diretório é destinado a armazenar recursos estáticos do projeto, como imagens, fontes, ícones, entre outros.

## Considerações Finais

A estrutura de diretórios definida visa proporcionar uma organização clara e eficiente do projeto. Ao separar as diferentes partes do projeto em diretórios distintos, facilitamos o desenvolvimento, manutenção e colaboração entre os membros da equipe.
